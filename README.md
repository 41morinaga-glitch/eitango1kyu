# 英検1級 英単語アプリ

シンプルなフラッシュカード式の英検1級単語学習アプリです。

## 機能

- フラッシュカード（タップで英⇄和を切替）
- 表示モード切替（英→和 / 和→英）
- スワイプ操作（左＝次へ、右＝前へ）
- 発音機能（Web Speech API、オン/オフ切替可）
- 自分で単語を追加・編集・削除
- JSON / CSV でのインポート・エクスポート
- データはブラウザの localStorage に保存

## 使い方

`index.html` をブラウザで開くだけで動作します。サーバー不要。

## ファイル構成

- `index.html` — 画面
- `style.css` — スタイル
- `script.js` — ロジック
- `words.js` — サンプル単語データ
